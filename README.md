# Reseñas a Maestros 📚⭐

Este proyecto es parte del **Laboratorio de Programación Web** de la **Facultad de Ingeniería Mecánica y Eléctrica** de la **Universidad Autónoma de Nuevo León**. El objetivo es desarrollar una plataforma web que permita a los estudiantes consultar y publicar reseñas sobre maestros, facilitando así la toma de decisiones académicas.

## 🚀 Objetivo del Proyecto

Crear un sitio web en el que los alumnos puedan:

- Compartir sus opiniones sobre maestros.
- Consultar experiencias previas de otros estudiantes.
- Calificar a los maestros y dar "me gusta" a reseñas útiles.
- Reportar comentarios inapropiados.
- Solicitar la inclusión de nuevos maestros o materias.

## 👥 Tipos de Usuarios

- **No Usuario (Visitante):**
  - Puede ver reseñas destacadas y buscar reseñas por maestro o materia.

- **Estudiante (Registrado):**
  - Puede dejar reseñas y calificaciones.
  - Puede dar "me gusta" o reportar reseñas.
  - Puede hacer peticiones de nuevas materias o maestros.

- **Administrador:**
  - Tiene acceso completo.
  - Puede visualizar reportes y aceptar/denegar peticiones.
  - Puede agregar nuevos administradores.

## 🧠 Tecnologías Sugeridas

- HTML/CSS/JavaScript
- Backend con PHP, Python o Node.js (según elección del equipo)
- Base de datos: MySQL, PostgreSQL o MongoDB

## 🖼️ Estructura del Sitio

- **Pantalla de Inicio:** Muestra reseñas destacadas por cantidad de "me gusta".
- **Pantalla de Búsqueda:** Permite buscar por materia o por maestro.
- **Pantallas de Reseña:** Se muestran todas las reseñas asociadas.
- **Pantalla de Peticiones:** Solo para estudiantes.
- **Pantallas de Inicio de Sesión y Registro:** Para autenticar usuarios.
- **Panel de Administrador:** Para gestionar usuarios y reportes.

## 📌 Requisitos

- Navegador moderno actualizado.
- Conexión a internet (si se implementa backend remoto).
- Cuenta registrada para funcionalidades completas.
